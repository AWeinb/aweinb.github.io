{ project } Win7 Logon

This was my latest project. It's a Windows 7 Logon editor with a lot of functionality. You are able to:
  • Change all images.
  • Change important sizes and margins.
  • Hide or show elements.
  • Change the used fonts.
  • Change important positions of elements.
  • ... and much more.
If you don't want to create new skins this is more interesting for you:
  • You can apply saved skins.
  • You can apply a backup. The backup is the logon state before you first started the program.
The program is written in Java (Version 7), so you will need this. I think it is relative user friendly and there should be no broken logon after you applied an already tested skin. You can try the program, just download it here:
